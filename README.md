# Anyblog

Anyblog is a simple website for discovering a random blog.

Press the button and it takes you straight to one blog from the current source pool.

## Where The Blogs Come From

Anyblog currently builds its blog list from:

- [Hacker News Popularity Contest](https://refactoringenglish.com/tools/hn-popularity)
- [Bear Blog newest discovery feed](https://bearblog.dev/discover/?newest=true)

The site also shows the latest fetched count for each source in the footer's `Sources` panel.
